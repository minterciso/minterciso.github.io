POC for migrating EBS applications and database to AWS private cloud

About

While working as a DBA/ATG with EBS, we create a POC for migrating over 60 instances of EBS (11 and 12.1 versions) from a private hosting solution, to the AWS cloud. We created the AWS infrastructure from the ground and, via a combination of Ansible, AWS Lambda and custom database replication scripts, we were able to execute this in less than 48 hours, with only a 2 hour window downtime.

Challenges

From designing the cloud infrastructure, based on a private hosting solution, to utilizing some more advanced tools to minimize the cost of the cloud infrastructure and increase only when needed, this was one of the biggest challenges (and one of the more fun ones) that I had to surpass until the time it came. With a good team of DBAs and extensive knowledge of Linux scripting, we were able to execute this flawlessly and reduce the cost (for the POC at least) of the infrastructure in around 30%.
